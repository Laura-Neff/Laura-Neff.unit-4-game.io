The Alignment Chart Game

Instructions

You will be given a random number at the start of the game

There are four versions of toothpaste below with different alignments.

There is a lawful good, a chaotic evil, a lawful evil, and a true neutral.

You win the game by matching your total score to the random number.

You lose the game if your total score goes above the random number.

The value of each toothpaste is hidden from you until you click on it.

Each time when the game starts, the game will change the values/alignments of each toothpaste.
